# Screenshots

Add your project screenshots to this folder.

## Suggested files

| Filename | What to capture |
|----------|----------------|
| `home.png` | Landing page / hero section |
| `scanner.png` | URL scanner input |
| `scan-result.png` | A completed scan result showing score and flags |
| `dashboard.png` | Dashboard with charts and scan history |
| `login.png` | Sign-in / sign-up modal |
| `mobile.png` | Mobile view of the scanner or dashboard |

## Tips

- Use a 1280×800 or 1440×900 browser window for consistency
- PNG format preferred for sharpness
- Capture both a safe URL scan and a suspicious/malicious one to show both states
